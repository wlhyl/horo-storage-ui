# 项目说明
该项目为[horo-storage-api](https://github.com/wlhyl/horo-storage-api.git)的ui。

# 开始使用

## 运行 horo-storage-api

根据 horo-storage-api 后端的 README.md 运行 API。

**注意**：API需要监听在8081端口。

## 安装依赖

```bash
npm install
```

## 启动项目

```bash
ng serve --open
```

# Docker 镜像构建

```bash
docker build -t horo/storage-ui .
```
